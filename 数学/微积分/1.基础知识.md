6. 习题总结
    1. 思考：基本不等式中 **$1$的用法**
        ![[Pasted image 20260202111503.png]]
        -  (1)左边有$A：ab+bc+ac$，而$(a+b+c)^2$可以出现$A$，因而替换右边的1然后平方，再往下证。**利用$1^2=1$**
        - (2)这里的思考是从基本不等式$a+b>=2\sqrt{ab}$中**开根号的便利性**出发的。**利用加法，两边同时加1**，左边的$1$用$a+b+c$替换
    2. 思考(2)：基本不等式 **地位**、**系数观察**
        ![[Pasted image 20260203085430.png]]
        - 地位(对称性)：$x\text{和}y$地位等价，可互相交换，所以$(1+\frac{1}{x})\text{和}(1+\frac{1}{y})$地位也等价。由此可以**猜测**$(1+\frac{1}{x})\text{和}(1+\frac{1}{y})$**两个都要放缩，且均>=3**
        - 系数观察：由上一步推理出$>=3$，**系数为3**，由此联想到$\sqrt[3]{abc}<=\frac{a+b+c}{3}$。而$(1+\frac{1}{x})=\frac{x+1}{x}$，此时分子只有两项，为了变成三项，结合已知$x+y=1$，将分子凑成$x+x+y$然后放缩，$(1+\frac{1}{y})$同理。相乘后得到结果
        - **解决疑问**：为什么$(1+\frac{1}{x})$要变成$\frac{x+1}{x}$而不是$x+y+\frac{1}{x}$？因为要先化为乘除主导再套入条件
            1. 对于具有**对称性**的放缩型不等式，优先识别**可重复的最小放缩单元**
            2. 若**可以通分**，先对该单元做等价的结构变形，从**加法**主导**转为乘除**主导，以增强整体性
            3. 在**结构明确**后，再利用已知对称条件凑项或对称化，最终施加不等式
    3. 思考：柯西不等式
        1. 可以把**平方和的乘积**转化为**乘积的和的平方**
            ![[Pasted image 20260203095720.png]]
        2. **1的用法**、**系数观察**
            ![[Pasted image 20260203100700.png]]
            - 1的用法：柯西不等式的左右都可以灵活运用1
                - 左：$(a+b+c)^2=(a\cdot 1+b\cdot 1+c\cdot 1)^2$
                - 右：$(a_1^2+a_2^2)=\frac{(a_1^2+a_2^2)\cdot (1^2+1^2)}{2}$
            - 系数观察：观察到不等式**右边系数为3**，推测出左边是**3项**乘积的和的平方；然后观察到右边有$6ab$，所以尝试将左边的$a+b$并为一项
    4. 思考：绝对值不等式的处理思路
        ![[Pasted image 20260203104052.png]]
        - 分类讨论法：不等式的形式简单明确时，直接分类求值
        - 利用三角不等式放缩：$|a|+|b|>=|a\pm b|$
            - 由于$a$可变，所以可能的图像有无限张
            - 对于任意一张图($a$确定)，利用三角不等式放缩，放缩后得到的**下界不含有$x$**。此时所有的图像有了**统一性**：即有一个只与$a$相关的下界$M$
            - 接下来只要**验证每张图都能取到M**，并保证$M>=4$
            - **验证"="成立性**：保证对于任意的$a$(特定图)，都有$x\text{使得}f(x)$都能取到$M$
            - 保证每张图(任意$a$)都能取到$M$后，只需从中筛选出$M>=4$的图，解得充要条件
    5. 思考：二次函数根分布的问题(证明的右半部分)
        ![[Pasted image 20260203115243.png]]
        1. **明确**不等式的**比较对象应该同类型**：不可能是一个自变量和一个函数值，由此把$x_1$先变为$f(x_1)$，从而转化为研究$f(x)$的单调性问题
        2. 给定区间为$(0,x_1)$，故研究$f(x)$对称轴$t=\frac{b}{-2a}\text{和}0、x_1$的关系
        3. 对$g(x)$使用韦达定理，同时$x_2<\frac{1}{a}$，从而得到$x_1>\frac{b}{-a}$
        4. 分类讨论：$t<=0$时显而易见；$t>0$时，由于$x_1>\frac{b}{-a}=2t>t$，所以对称轴在$0、x_1$之间，根据$x_1、0$**与对称轴的距离**得证
