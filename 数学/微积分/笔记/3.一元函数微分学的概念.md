# 2.结论 
1. $f(x)$在点$x_0$可导的充要条件：左右导数均存在且相等(左右导数即单侧导数)
    ![[Pasted image 20260306185602.png]]
2. $f(x)$在点$x_0$可导，则在$x_0$必连续。反之不成立(反例：$|x|$在$x=0$处)
    ![[Pasted image 20260306190021.png]]
    1. 可以结合导数定义的函数式，从收敛速度来理解：连续时仅仅要求分子趋于0，但是**导数要求**分子趋于0的**速度不低于分母**。故无法由连续推导出可导
    2. 某一点可导只能推导出该点连续，连附近点的连续性都保证不了，更谈不上推出附近的点可导
    3. 推广到高阶导数
        ![[Pasted image 20260306205429.png]]
3. **$f(x)与|f(x)|$在连续和可导上的关系**
    ![[Pasted image 20260307093820.png]]
    1. 连续的微观理解
        ![[Pasted image 20260306200429.png]]
        ![[Pasted image 20260306200446.png]]
    2. 可导的微观理解
        $f(x_0)=0\text{且在}x_0\text{处可导时}$：
        1. $f'(x_0)\neq 0$时，加绝对值之后必然导致左右导数不相等，故$|f(x)|在x_0$处不可导
        2. $f'(x_0)=0$时：函数式中分子的收敛速度远大于分母，此时$|f(x)|在x_0$处收敛速度依然保持远大于分母，故必定可导，且导数值为0    
        ![[Pasted image 20260306203127.png]]
        $f(x_0)\neq0\text{且在}x_0\text{处可导时}$：利用极限的保号性框住一个小区间内的点，然后进行翻转(如图)，易知导数存在
        ![[Pasted image 20260306203138.png]]
4. $f(x)在点x_0$切线存在与导数存在的关系
    ![[Pasted image 20260306204117.png]]
    1. 导数存在切线必定存在
    2. 导数不存在但为无穷导数时，切线也存在。$y=x^ \frac{1}{3}在x=0$处
    3. 左右导数存在且不等时，切线不存在。$y=|x|$
5. $f(x)在x_0$处可微的充要条件
    1. 可导必可微
