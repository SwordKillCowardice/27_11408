# 1.知识点
1. 本节总览：三点(极值点、最值点、拐点)、两性态(单调性、凹凸性)、一线(渐近线)
2. 理解极值
    ![[Pasted image 20260311163409.png]]
    1. 极值是局部概念，只要求$x_0$附近(某邻域内)
    2. 邻域是双侧邻域，左右两边都得有定义(只比一边无法确定是极值)。故而在区间端点处不讨论极值(也不讨论间断)
    3. 极值点只是一个横坐标
    4. 常数函数处处是极值(既是极大值，又是极小值)(从定义理解)
3. 判别单调性
    ![[Pasted image 20260311173813.png]]
    1. 导数为$0$不意味着点水平分布，而只代表函数值从两侧逼近当前点时发生的变化无穷小。如$y=x^3在x=0处导数为0$，但是左边的值都比他小，右边的值都比他大

# 2.结论
1. 间断点可以是极值点：采用挖点构造即可
    ![[Pasted image 20260311163719.png]]
2. **极值点判别的一必要三充分**
    1. 一阶可导点是极值点的必要条件
        ![[Pasted image 20260311183620.png]]
        1. 证明：若$f'(x_0)\neq 0$，则取微观区间必有$x_0$不是极值点
        2. **极值点要么是驻点，要么是不可导点**. 找极值点时就找这两种点
            ![[Pasted image 20260311183715.png]]
    2. 判别极值点的充分条件
        1. 第一充分条件：点连续，附近导数值异号
            ![[Pasted image 20260311184309.png]]
        2. 第二充分条件：点二阶可导，一阶为$0$且二阶不为$0$
            ![[Pasted image 20260311184348.png]]
            - 用二阶导定义+保号性证明
                ![[Pasted image 20260311184423.png]]
        3. 第三充分条件：点$n(偶数)$阶可导，前$n-1$阶为$0$，第$n$阶不为$0$(会用就行，不用推导)
            ![[Pasted image 20260311184445.png]]
3. **拐点判别的一必要三充分**(类比极值点)
    1. 二阶可导点是拐点的必要条件
        ![[Pasted image 20260311201522.png]]
        拐点要么二阶导为0，要么二阶不可导
        ![[Pasted image 20260311201603.png]]
    2. 判别拐点的充分条件(注意拐点定义只需要一阶导存在，没有要求其取值)
        1. 第一充分条件：点连续，附近二阶导异号
            ![[Pasted image 20260311201635.png]]
        2. 第二充分条件：点三阶可导，二阶为0且三阶不为0
            ![[Pasted image 20260311201707.png]]
        3. 第三充分条件：点$n(奇数)阶$可导，$2-(n-1)$阶为0，第$n$阶不为0
            ![[Pasted image 20260311201834.png]]
