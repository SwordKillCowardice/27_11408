# 1.三角函数
1. ==核心理解==：$sinx、cscx、tanx$分别与$cosx、secx、cotx$**高度对称**，这意味着他们复合而成的函数是**高度可交换**的，**求解思路也可以迁移**
2. $sin^mx\cdot cos^nx$型 ($m>=1$)
    1. **对称性**：该题型的思路可以直接迁移到$cos^mx\cdot sin^nx$型 ($m>=1$)
    2. $m\text{为}奇数$：凑$dcosx$，本质是幂函数
        1. $m=1$：直接凑

        2. $m>=3$：凑一个$dcosx$，剩下的写成$(1-cos^2x)^{\frac{m-1}{2}}$
            ![[Pasted image 20260208082610.png]]
            ![[Pasted image 20260208083611.png]]
            ![[Pasted image 20260215113012.png]]
    3. $m=2$：$sin^2x$变形为$1-cos^2x$，本质是求$cos^nx-cos^{n+2}x$
        ![[Pasted image 20260208081418.png]]
        ![[Pasted image 20260208083739.png]]
3. $\frac{1}{sin^mx\cdot cos^nx}$型 ($m>=1\text{且}n>=1$)
    1. **通用思路**：利用$sin^2x+cos^2x=1$反复分解降幂
    2. $m\neq n$：使用分解降幂法

    3. $m=n$：优先使用$sinxcosx=\frac{sin2x}{2}$对分母进行变形，转化为求$csc^mx$ ; 也可以使用分解降幂法

4. $\frac{f(x)}{1\pm sinx}$型
    1. **对称性**：该题型的思路可以直接迁移到$\frac{f(x)}{1\pm cosx}$型，但要**注意结构交换**，比如$\frac{secx}{1+sinx}$应该对应$\frac{cscx}{1+cosx}$
    2. **通用思路**：分子分母同乘$1\mp sinx$，将分母化为$cos^2x$
    3. **扩展性**：该题型的思路可以扩展到$\frac{f(x)}{({1\pm sinx})^2}$型和$\frac{f(x)}{(1\pm cosx)^2}$型，分子分母同乘$(1\mp sinx)^2$，将分母化为$cos^4x$

    4. **特殊性**：$\frac{f(x)}{1\pm cosx}$型和$\frac{f(x)}{(1\pm cosx)^2}$型还可以**利用二倍角公式**$cos2x=2cos^2x-1$和$cos2x=1-2sin^2x$进行变形

# 2.有理分式积分($\int\frac{P(x)}{Q(x)}\,dx$)
1. ==核心理解==：任何一个有理分式，最终都可以拆分为**整式+分式展开式**
    ![[Pasted image 20260215091914.png]]
2. **通用思路**：
    1. 能约分先约分
    2. 真分式直接求；假分式(**分子次数不低于分母**)先化为整式+真分式
    3. **求真分式积分**：先求其分式展开式，再对展开式中各项逐个积分
        ![[Pasted image 20260215073615.png]]
        1. 对分母进行因式分解，分解为第一列的四种形式
        2. 分母的每个因子对应的拆分项如第二列，拆分项之和即分式展开式
        3. 对各个拆分项逐个求积分
3. 例题
    1. 假分式：先把分子写成$P(x)\cdot \text{分母}+C$(即拆分)，然后分别求
        1. 简单情况

        2. 使用平方差公式
            ![[Pasted image 20260215093417.png]]
        3. 使用立方和公式

        4. 使用多项式除法：**缺项要先补齐**：$t^4+0t^3+0t^2-t+0$；**余数的次数必须严格小于除数的次数**
            ![[Pasted image 20260215094445.png]]
    2. 真分式：简单则直接观察得之，复杂则求其分式展开式
        1. 简单情况：直接把分子改写成分母线性组合
