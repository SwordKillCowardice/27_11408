## 1.1 数据结构的基本概念
1. 知识总览
    ![[Pasted image 20260309214350.png]]
2. 基本概念(了解)
    1. 数据是信息的载体
    2. 数据元素是数据的基本单位，数据项是构成数据元素的最小单位
        ![[Pasted image 20260309211242.png]]
    3. 数据对象是具有相同性质的数据元素的集合
    4. 数据类型：值的集合和定义在此集合上的一组操作
        ![[Pasted image 20260309220015.png]]
    5. 抽象数据类型(ADT)：抽象数据组织(逻辑结构)+操作(运算)，等价于定义了一个逻辑结构
    6. 关键字：可以唯一区分数据元素
3. **数据结构**(掌握)
    1. 即彼此之间**具有特定关系**的数据元素的集合。结构指元素之间的关系
        ![[Pasted image 20260309215054.png]]
    2. **数据结构三要素**
        1. 逻辑结构：数据元素之间的逻辑关系
            ![[Pasted image 20260309215257.png]]
            1. 集合：各个元素同属一个集合，别无其他关系
            2. 线性：一对一
            3. 树形：一对多
            4. 图状：多对多
        2. 物理结构(存储结构)：用计算机具体实现
            ![[Pasted image 20260309215510.png]]
            1. 顺序：逻辑上相邻物理上也相邻
            2. 链式：逻辑上相邻物理上可以不相邻
            3. 索引：建立索引表，每个表项称为索引项(关键字+地址)
                ![[Pasted image 20260309215738.png]]
            4. 散列：哈希表
        3. 数据的运算(定义+实现)：运算的定义针对的是逻辑结构，指出运算的功能；而实现针对存储结构，指出运算的具体步骤
        4. 确定逻辑结构和运算就是定义了一个数据结构，而确定存储结构用于实现数据结构
        5. 理解：顺序存储需要分配大块的连续空间，而非顺序不用(第二点)
            ![[Pasted image 20260309215828.png]]
