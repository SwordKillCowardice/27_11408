# 三、数据链路层
1. 循环冗余校验码(基本思想)：实现约定除数(生成多项式的系数)，传输被除数，看余数是否还为0。被除数=数据+校验位(校验位位数与生成多项式次数相同)
    ![[Pasted image 20260210190326.png]]
2. 模2除：位数对齐，被除数第一位是1就商1，是0就商0。模2减就是异或运算

5. *CSMA/CD*协议：1-坚持(先听后发)、边听边发、冲突停发(发送时如果检测到冲突，立刻停止发送)、随机重发(根据冲突次数来决定是否等待，等多久)
    ![[Pasted image 20260301204514.png]]
    1. 争用期：争用期内不冲突，信道就归发送方了
    ![[Pasted image 20260301205147.png]]
    2. 最短帧长：发送方单次发送的帧应该要持续整个争用期，否则会无法判断发送的帧是否遇到冲突；如果实际数据很少，需要填充至合法长度再发
    ![[Pasted image 20260301205831.png]]
    ![[Pasted image 20260301210031.png]]
    3. *CSMA/CD*没有*ACK*机制，如果发送时未检测到冲突，就认为发送成功
    4. 最长帧长：避免节点一直占领信道
    5. 以太网使用CSMACD，并规定最短帧长64B，最长1518B
