# 一、计算机网络体系结构
1. 一些交换设备
    1. 集线器：会发生数据冲突。同一时刻只有一台电脑可以发数据
        ![[Pasted image 20260128211036.png]]
    2. 交换机：解决了数据冲突。交换机之间也可以彼此连接
        ![[Pasted image 20260128211304.png]]
    3. 家用路由器=**路由器+交换机**。交换机部分把家里的平板、手机等连成一个子网，路由器部分负责将子网与外界网络连接。
        ![[Pasted image 20260201214012.png]]
2. 性能指标
    1. 带宽(家庭接入示意图)：外链路直接连到光猫，光猫和家用路由器之间有网线。限制带宽的有：外链路带宽、光猫和路由器之间的网线带宽、路由器WAN口最高速率
        ![[Pasted image 20260201223259.png]]
    2. 吞吐量：网线的吞吐量=发送信道吞吐量+接收信道吞吐量、网卡的吞吐量=接收吞吐量+发送吞吐量、网络总吞吐量=所有发送+接收吞吐量
        ![[Pasted image 20260201224100.png]]
